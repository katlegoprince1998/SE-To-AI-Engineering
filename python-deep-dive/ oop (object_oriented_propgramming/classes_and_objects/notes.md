## Classes and Objects

# How to create a class
  >> class keyword in python is used to create a class
  >> A class needs to be noun
  >> Use the keyword followed by a descriptive noun word to create the class

# How to create an object
  >> Define an object identifier and assign it the name of your class. 
  >> You can create multiple objects from one classes 

# The Pass Statement
  >> Class definition cannot be empty
  >> if your class definition happens to be empty, you can use "pass" to avoid getting an error
