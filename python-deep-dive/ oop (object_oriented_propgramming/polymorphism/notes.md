## Polymorphism

# What is polymorphism
  >> This word means many forms
  >> A method/function/operator with the same name can be executed on many objects or classes.

# Example
  >> len()
  >>> In strings it can return the number of characters.
  >>> In a Tuple it can return the number of items in a Tuple collection
  >> In a dictionary it can return the number of key-value pairs
