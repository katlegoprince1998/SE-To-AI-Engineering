## OOP (Object Oriented Programming)

# What is OOP ??
  >> OOP Stands for Object Oriented Programming
  >> Python is an object oriented language
  >> Allows organization of code in a form of objects for better reusability.

# Advantages of OOP
  >> Better code organization
  >> Helps follow the DRY principle
  >> Assist in writing reusable applications
  >> Makes code easier to maintain, debug and reuse

# What are Classes and objects ?? 
  >> A class is a template/blueprint for an Object
  >> An object is an instance of a class
