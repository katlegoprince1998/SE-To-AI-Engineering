## File handling

>> This is an important part of any web application

# open() function
  >> This is the key function for working with files
  >> It takes two parameters - file name and mode
  
# Four methods (modes) for opening a file
  1. "r" - Read - Default value - opens the file for reading, error if the file does not exist
  2. "a" - Append - Opens a file for appending, creates the file if it does not exist
  3. "w" - Write - Opens a file for writing, creates the file if it does not exist
  4. "x" - Create - Creates the specified file, returns an error if the file exists
  --- Additional modes
      1. "t" - Text - Default value - text mode
      2. "b" - Binary - Binary mode (e.g images)
